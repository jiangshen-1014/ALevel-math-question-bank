# 数学题库 · Edexcel & CIE Question Bank

一个本地运行的 A-Level 数学题库与组卷工具，无需安装、无需服务器，双击 `index.html` 即可使用。

---

## 一、快速开始

直接双击打开 `index.html`（推荐 Chrome / Edge）。
> 公式渲染依赖 MathJax（首次打开需联网加载，之后浏览器会缓存）。

---

## 二、核心功能

### 1. 分类浏览（大类 → 科目）
左侧栏先分两大类：
- **CIE**（剑桥）：9709 的 P1/P2/P3/M1/S1/S2，9231 进阶的 FP1/FP2/FM/FS
- **Edexcel**（爱德思 IAL）：P1–P4、M1–M3、S1–S3、FP1–FP3、D1

点击科目即可筛选；顶部可按 **章节 / 难度** 二次筛选，或直接搜索题干、考点、来源。

### 2. 题目录入（三种方式）

**方式 A：截图自动录入（推荐）—— 标准 4 段式指令**

发给我时按以下 4 部分提供（顺序固定）：

1. **考试局 / 科目**：`Edexcel/FP2`（大类/科目）
2. **真题来源**：`26_Jan_01` = 2026年 · 1月(Jan) · 卷号01（`年_月_卷号`；卷号可带字母，如 `26_Jun_01A`）
3. **题干内容截图**：我会把它转换为 LaTeX 文本并生成解析 —— **此截图本身不入库**，只保留转好的文字
4. **配图截图**（可选）：如坐标图、力学图。**有则原样入库**；不提供则该题无配图

我会自动完成：题干转 LaTeX → 打章节/考点标签 → 写中英结合详细解析 → 解析真题来源（年份/月份/卷号）→ 写入题库。

> 示例指令：`Edexcel/FP2` + `26_Jan_01` + [题干截图] + [配图截图]

**方式 B：手动新增**
点右上「＋ 新增题目」：
- 选大类 / 科目，章节有预设下拉（P1、FP1、S1、M1 等）
- 正文与解析支持 LaTeX：行内 `$...$`、独立公式 `$$...$$`、`**加粗**`，右侧实时预览
- 配图可点击上传，或在弹窗内直接 **Ctrl+V 粘贴截图**

**方式 C：批量导入 Excel / CSV（4 列表格）**
把多道题按下面 4 列（外加可选列）填进一张表，一次导入：

| 列 | 字段 | 说明 |
|---|---|---|
| A | **考试局/科目** | `Edexcel/FP2` 或 `CIE P1`（支持「/」或空格） |
| B | **来源** | `26_Jan_01A_3`（年_月_卷号[_题号]；不带题号也可） |
| C | **题干** | 两种填法：① 填**截图文件名**（如 `Q3_stem.png`），连图发我转 LaTeX；② 直接填 **LaTeX / 文本** |
| D | **配图** | 截图文件名或留空（无配图）；直接填 LaTeX 时可留空 |
| E–I | 章节 / 考点 / 难度(1–5) / 分值 / 解析 | 选填；解析留空则由我补写 |

- **应用内自助导入**：题干已是 LaTeX/文本时，点右上「**批量导入**」选 `.xlsx/.xls/.csv` 即可（首行表头自动识别中英文；按 id 去重，重复导入不新增）。
- **连截图发我批量录入**：题干/配图列填的是截图文件名时，把 Excel + 这些图片一起发我，我读表后逐题转 LaTeX、写解析、一次性入库（与 4 段式等效，只是批量处理）。
- 已附 `批量录入模板.xlsx` 可直接套用。

### 3. 组卷（可选带/不带解析答案）
- 在题卡点「＋ 加入组卷」，右上「🧺 组卷篮」查看
- 可上移/下移调整题序、显示总题数与总分
- 用开关切换 **含解析（教师版）/ 不含解析（学生版，留答题空白）**
- 填试卷标题 → 「生成试卷」→「🖨️ 打印 / 导出PDF」（浏览器打印即可存 PDF）

### 4. 数据管理
- **批量导入**：见上文「方式 C」，支持 4 列表格一次导入多题
- **导出**：把整个题库存成 JSON（含 base64 配图），可备份/迁移
- **导入**：合并 JSON 到当前题库（按 id 去重）
- 数据默认存在浏览器 localStorage，请定期导出备份

## 三、题目数据结构（供参考）

```js
{
  id, board,          // "CIE" | "Edexcel"
  subject,            // "P1" / "FP1" / "S1" ...
  chapter,            // 章节
  topics: [],         // 考点标签
  difficulty,         // 1–5
  marks,              // 分值
  source,             // 来源（可选）
  stem,               // 题目正文（LaTeX 文本）
  figure,             // 配图（base64 或相对路径，可空）
  solution            // 详细解析（中英结合）
}
```
